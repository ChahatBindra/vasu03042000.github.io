  1. 1. Click On the Simulation tab from the left dropdown menu.
  2. 2. Then Click on "Click Here" to go to the default Simulation window which will be hub.
  3. 3. You will be directed to a new tab.
  4. 4. Default is Hub but still you can start any simulation first by clicking on the buttons visible to you.
  5. 5. In the Simulation Frame, select the Source PC's Mac Address and the Destinaton PC's Mac Address.
  6. 6. Click on the start Simulation to start the simulation.
  7. 7. Simulation starts.
  8. 8. Try out the other devices just by clicking on the button visible to you stating ROUTER, HUB, SWITCH, BRIDGE.
  9. 9. If you want to exit the simulation just close the tab.
